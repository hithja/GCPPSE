# GCPPSE
GCPPSE is my SFML 2D engine
# What can this engine do?
Small 2D Platformer games with bugs, but in future i'm going to add more features!
# How to build?
Install `SFML`:
- RHEL/Fedora-based
```
sudo dnf install SFML SFML-devel
```
- Debian/Ubuntu-based
```
sudo apt install libsfml-dev
```
- Arch Linux
```
sudo pacman -S sfml
```
Then build it:
```
./build.sh
```
